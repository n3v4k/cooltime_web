# 쿨타임 (CoolTime)

체감온도 기반 폭염 작업중지·휴식 의무를 실시간으로 확인하고 기록하는 웹 서비스입니다.
`docs/requirements_v3.md`, `docs/design-guide.md` 요구사항을 기반으로 구현했습니다.

## 실행 방법

1. Supabase에서 `database/schema.sql`을 SQL Editor에 실행해 테이블을 생성하세요.
2. `.env.local`의 Supabase 값은 이미 채워져 있습니다. 실시간 날씨 연동을 원하면
   [OpenWeatherMap](https://openweathermap.org/api)에서 API 키를 발급받아
   `OPENWEATHER_API_KEY`에 입력하세요. (미입력 시 관리자 화면에서 체감온도를
   슬라이더로 수동 입력하는 폴백 모드로 동작합니다.)
3. 의존성 설치 후 개발 서버 실행:

```bash
npm install
npm run dev
```

4. `http://localhost:3000` 접속 → 홈에서 작업자/관리자 모드 선택
   - 관리자 모드에서 먼저 현장을 생성해 4자리 코드를 발급받으세요.
   - 작업자 모드에서 그 코드로 참여하면 실시간으로 상태가 동기화됩니다 (5~10초 폴링).

## 주요 기능

- 위치 기반(또는 수동 입력) 체감온도 → 5단계 신호등 배지 + 휴식 규칙 카운트다운
- 옥외/실내 토글 (오후 2~5시 특례 반영)
- 온열질환 자가진단 3문항 → 열사병 의심 시 즉시 119 안내, 결과는 관리자 화면에 실시간 반영
- 관리자 대시보드: 작업 중/휴식 중/휴식 초과 요약, 긴급 배너, 작업자별 휴식 시작, 오늘 기록 CSV 내보내기
